# Node.js basics
To check, you need to switch to the '**nodejs-basics**' branch:   
1. git clone https://github.com/rahunak/node-nodejs-basics.git
2. git checkout nodejs-basics
3. open package.json
4. run scripts using commands or directly from the folder
